About This Library

This library has been created using purely front-end technology. It uses local storage data on your browser to ensure a persistent library across sessions.

This data is not sent to any third parties or even myself, it is purely to enhance your experience and make sure your time is not wasted.

When using the autofill function on the book form, a call is made to Google Books API which returns the first match it finds when searching by relevance
